Welcome to Simply Maple

Simply Maple is a MapleStory website/cms used for MapleStory Private Servers. This CMS includes a home, register, vote, downloads, donate, forums, and chat pages! Each page includes different content in the body of the page (AKA the base or center).

The Design

The design was made 100% with CSS3. (The background is a separate image). The design is based on a Metro Style, all of the colors make a very sleek looking Flat UI. The main colors are Black and White. With the background being a majestic colored forest.

The Team

Well, the only person on the team is the author of the website, SilentThief, AKA Hexish. This is a solo development.

Code

This site was coded with PHP, HTML5, CSS3, and JavaScript. The main point of the website code, is to make it clean and easy to read. This is a new experiment for the team, making it 100% more efficient.
